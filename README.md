# Stock-trend-analyzer
A python project to analyze stock trends using yfinance and pandas
##How to Run 
1. Clone the repo.
2. Install dependencies: pip install yfinance pandas matplotlib
3. Run 'jupyter notebook' and open 'stock_analyzer.ipynb'.
